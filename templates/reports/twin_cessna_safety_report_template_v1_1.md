## 2.2 Coding Framework (Taxonomy v1.1)

Each event coded with 1–3 primary contributing factors from the following categories:

### Human / Cognitive
- Normalization of deviance
- Assumption of performance
- Plan continuation bias
- Checklist non-compliance
- Task saturation

### Maintenance / Mechanical
- Aging-aircraft degradation
- Incomplete troubleshooting
- Deferred discrepancy normalization
- Maintenance-induced failure

### Performance / Operations
- Weight & balance misjudgment
- Density altitude underestimation
- Single-engine performance assumption
- Icing performance degradation

### Experience-Related
- Low time in type
- High time / complacency risk
- Lapsed recency
- Informal or incomplete training
